# 01. Project Overview

## Core Vision
This project is a web-based prototype of a "Block Blast!" style puzzle game. The primary design philosophy is **Zen and Minimalist**. It is built for casual players seeking a relaxing, stress-free experience, actively avoiding "tryhard" mechanics like timers, aggressive scaling difficulty, or harsh penalties.

## Target Audience
Casual gamers, people looking to unwind during a break, or anyone who enjoys satisfying spatial puzzles without the pressure of a ticking clock.

## Key Objectives
- **Simplicity:** Clean UI with zero clutter. 
- **Forgiving Gameplay:** No time limits. The game ends only when no moves are possible, but the framing should be "restful" rather than a harsh "GAME OVER."
- **Satisfying Feedback:** Gentle visual and audio cues for placing blocks and clearing lines.
- **Web-First:** Optimized for modern browsers and deployed easily via GitHub Pages.
