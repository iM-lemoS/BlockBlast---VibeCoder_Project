# 02. Game Mechanics

## The Grid
- An 8x8 grid.
- Minimalist styling (e.g., soft rounded squares, muted background colors).

## Shapes
- Players are presented with 3 random shapes at a time in a "tray" below the grid.
- Shapes are composed of 1 to 5 blocks (similar to Tetrominoes but simpler).
- Shapes are dragged and dropped onto the grid.
- Once all 3 shapes are placed, 3 new shapes are generated.

## Clearing Lines
- If a full row or column is completed, it clears from the board.
- Multiple lines cleared at once yield a higher score, accompanied by a soothing chime.

## End State
- The game concludes when none of the available shapes can fit anywhere on the grid.
- Instead of a jarring "Game Over", the game offers a gentle "Take a breath, try again" prompt, saving the high score.
