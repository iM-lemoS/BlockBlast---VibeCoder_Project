# 06. Technical Stack

- **Framework:** React 18 (Functional components & Hooks).
- **Build Tool:** Vite (Much faster and lighter than Create React App).
- **Routing:** `react-router-dom` (HashRouter).
- **Styling:** Tailwind CSS or CSS Modules. (Tailwind is recommended for quickly prototyping minimalist, pastel color palettes and rounded UI elements).
- **State:** React `useReducer` / Context.
- **Drag and Drop:** `@dnd-kit/core` (Provides excellent accessible and mobile-friendly drag-and-drop hooks).
- **Deployment:** `gh-pages` npm package.
